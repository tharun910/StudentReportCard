# Student Report Card Generator

## Description
A Python utility that reads student data from a CSV file, calculates averages, standard deviations, assigns grades, and generates report cards.

## Features
- Reads student data from CSV
- Calculates average marks
- Calculates standard deviation
- Assigns letter grades
- Generates individual report cards

## Technologies Used
- Python
- Git
- GitHub

## How to Run

1. Open terminal
2. Run:

python report_card_generator.py

## Output

Report cards are generated inside the report_cards folder.
